Message

仕事にかける想い

自分の好奇心・成長意欲でビジネスを支える

WorkFlow

制作の流れ

ご相談〜納品までのおおまかな流れです。状況に応じて柔軟に対応させていただきます。

STEP01

ご相談・お見積もり

要件や納期、予算など当サイトのお問い合わせページより お気軽にお問い合わせください。 対応内容に応じてお見積もりをお送りいたします。

STEP02

コーディング制作

頂いたのでデザインデータを元に制作を開始します。 コーディング後は品質チェックを行います。

STEP03

確認

お客様に制作したサイトに問題がないかご確認いただきます。 修正・変更があれば対応を致します。

STEP04

納品 / サイト公開

作成したサイトのサーバーアップロード もしくはファイル一式の納品どちらでも対応可能です。 本番サーバーアップ後再度確認をいただき、 問題なければ納品完了となります。
